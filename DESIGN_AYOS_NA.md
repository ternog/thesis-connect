## 🚀 Paano I-apply?

### Step 1: Clear Browser Cache
```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

### Step 2: Restart Frontend (kung tumatakbo)
```bash

npm start
```

### Step 3: Check ang Result
- Open sa browser
- Test sa different screen sizes
- Verify na organized na ang layout
